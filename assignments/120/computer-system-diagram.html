How Components Work Together:

1. Power Supply provides electricity to all components

2. CPU fetches instructions from Storage and loads them into RAM

3. CPU processes data and instructions from RAM

4. Results are sent to Output devices or stored back in Storage

5. Motherboard facilitates all communication between components
